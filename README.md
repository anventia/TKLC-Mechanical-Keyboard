# Tenkeyless-Compact
A Tenkeyless (TKL) Mechanical Keyboard, but more compact.
- PCB custom-made in KiCAD
- Support for:
  - Standard Cherry-style hotswappable switches
  - Gateron Low Profile hotswappable switches + Kailh Choc LP Stabilizers
- 3D printed plate + case
